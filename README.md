

# 🎬 视频转录工具 - GPU加速版

一个基于 Whisper 技术的视频语音转录工具，支持本地视频和在线视频（YouTube/Bilibili）转录为文字文档，具备 GPU 加速功能。

## ✨ 核心特性

### 🚀 高性能转录

- **GPU 加速**：支持 NVIDIA GPU 加速，提升转录速度 5-10 倍
- **多精度模式**：自动适配 float16/int8/float32 计算模式
- **批处理优化**：智能批处理，最大化 GPU 利用率

### 📁 多来源支持

- **本地视频**：支持 MP4、MKV、AVI、MOV 等常见格式
- **在线视频**：支持 YouTube、Bilibili、抖音等平台
- **会员视频**：支持通过 Cookie 文件登录下载会员内容

### 🎯 智能功能

- **多语言识别**：支持中文、英文、日文、韩文等 99 种语言
- **静音检测**：VAD 技术自动过滤静音片段
- **实时监控**：GPU 使用率、温度、VRAM 占用实时显示
- **性能测试**：内置 GPU 性能测试工具

### 📊 专业输出

- **多种格式**：支持 SRT 字幕文件 + TXT 纯文本文件
- **时间戳**：精确到毫秒的时间标注
- **批量处理**：支持批量转录多个视频文件
- **自定义输出**：灵活配置输出目录和文件名
